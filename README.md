# Amazon Price Tracker Bot

The Amazon Price Tracker Bot is a Python-based automation tool designed to monitor product prices on Amazon and maintain a historical record of price changes over time. Built using Selenium for web automation and Pandas for data handling, the bot programmatically navigates product pages, extracts current pricing information, and stores it in an Excel file along with timestamps for tracking trends. The system can also compare the latest price with previously recorded values and trigger email notifications when a price change is detected, helping users stay informed about potential deals or fluctuations.

The bot operates in headless mode, allowing it to run efficiently in the background without requiring a visible browser interface. It supports tracking multiple products simultaneously by maintaining a configurable list of product URLs, making it flexible and scalable for different use cases. Additionally, it logs execution details to ensure transparency and easier debugging. This project is particularly useful for users who want to automate price monitoring and avoid manually checking product listings, while also serving as a practical implementation of web scraping, automation, and basic data pipeline concepts in Python.
